# etm to tklr

## itemtypes

| type    | etm | tklr |
| ------- | --- | ---- |
| event   | \*  | \*   |
| task    | -   | ~    |
| project | -   | ^    |
| job     | @j  | @~   |
| note    | %   | %    |
| inbox   | !   | ?    |
| goal    | ~   | +    |

## changes

etm json -> tklr

- minutes: &n -> &M
- hours: &h -> &H
- months: &M -> &m (monthdays stays &d)
